Текстовая приключенческая игра "Лабиринт сокровищ", где игрок исследует комнаты, решает загадки, собирает предметы и пытается найти сокровище.

# Установка и запуск

## Требования
- Python 3.12 или выше
- Poetry (менеджер зависимостей)

## Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/ageeva139/project1_Ageeva_Alina_M25-555.git
cd project1_Ageeva_Alina_M25-555

# Установите зависимости
make install
# или
poetry install
```

## Запуск
```bash
make project
# или
poetry run project
```

# Описание проекта

## Цель игры
Найдите rusty_key, чтобы попасть в комнату сокровищ (treasure_room), затем используйте treasure_key или решите загадку, чтобы открыть сундук и победить

## Комнаты лабиринта
entrance - Вход в лабиринт
hall - Главный зал с загадкой
trap_room - Комната с ловушками
library - Библиотека с древними свитками
armory - Оружейная комната
treasure_room - Комната с сокровищами (финал)
garden - Тайный сад
secret_room - Секретная комната

## Основные команды
look - Осмотреться вокруг
inventory - Показать инвентарь
take <предмет> - Взять предмет
use <предмет> - Использовать предмет
solve - Решить загадку
go <направление> - Перейти в направлении
help - Показать справку
quit или exit - Выйти из игры
